# Robot Kinematics Simulator (2D 3-DOF)

A simple Python project that demonstrates analytical forward and inverse kinematics for a 2D 3-DOF planar robot arm, with interactive visualization.

## Files
- `forward.py`: Forward kinematics + visualization
- `inverse.py`: Inverse kinematics + interactive target selection

## How to run
```bash
python forward.py
python inverse.py
